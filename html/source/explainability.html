explainability.calculate_gradient(model, interpolated_input, actions, feature_idx, stock_idx, stock_dimension, tech_indicator_list, h=0.1)¶

Calculate gradient of the loss function w.r.t. the input features.

Parameters

  • model (torch.nn.Module) – the model to be used for the gradient calculation

  • interpolated_input (numpy.ndarray) – the input features to be used for the gradient calculation

  • actions (numpy.ndarray) – the actions to be used for the gradient calculation

  • feature_idx (int) – the index of the feature to be used for the gradient calculation

  • stock_idx (int) – the index of the stock to be used for the gradient calculation

  • stock_dimension (int) – the dimension of the stock to be used for the gradient calculation

  • tech_indicator_list (list) – the list of technical indicators to be used for the gradient calculation

  • h (float) – the step size to be used for the gradient calculation

Returns

the gradient of the loss function w.r.t. the input features

Return type

torch.Tensor
